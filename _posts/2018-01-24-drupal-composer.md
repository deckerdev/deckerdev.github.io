Composer + Drupal 8 = Awesome

## Common commands

Check for outdated:
```bash
composer outdated
```

Update Drupal core:
```bash
composer update drupal/core --with-dependencies
```

Update single module:
```bash
composer update drupal/MODULE_NAME --with-dependencies
```
